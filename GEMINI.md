# Project Context: Property Manager

## Overview
A property management dashboard starting with a mortgage payoff calculator, evolving into a full-stack application.

## Architecture
- **Frontend:** React + TypeScript + Tailwind CSS
- **Backend:** Firebase (Auth, Firestore, Functions)

## History
- Started as a single HTML file (`MortgageCalc-v1.6.html`).
- Added basic Tailwind setup.
- Refactoring to full-stack React app (Sprint 3).
